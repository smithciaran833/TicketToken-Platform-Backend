# API Documentation

## What Goes Here

- **API Gateway Routes**: All exposed endpoints (what users/frontend hit)
- **Service APIs**: Internal service-to-service APIs
- **Authentication**: How auth works, JWT structure, token refresh
- **Rate Limiting**: Rate limit rules and bypass tokens
- **Error Codes**: Standard error responses across all services
- **Request/Response Examples**: Real examples for each endpoint

## Files to Create

- `gateway-endpoints.md` - All public API endpoints
- `authentication.md` - Auth flow and token management
- `internal-apis.md` - Service-to-service communication
- `error-handling.md` - Error codes and responses
- `examples/` - Directory of request/response examples
